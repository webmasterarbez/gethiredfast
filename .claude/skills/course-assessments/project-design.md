## Project Types

### Capstone Project
Comprehensive application of course learning.
- Spans multiple modules
- Requires synthesis
- Portfolio-worthy output

### Mini-Project
Focused application of specific skills.
- Single module scope
- Quick feedback cycle
- Iterative improvement

### Peer-Reviewed Project
Learning through review and feedback.
- Submit → Review others → Revise
- Rubric-guided feedback
- Community learning

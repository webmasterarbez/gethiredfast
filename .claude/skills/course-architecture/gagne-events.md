# Gagn√©'s 9 Events of Instruction

Use these events to structure each lesson for maximum learning effectiveness.

## The 9 Events

| # | Event | Purpose | Examples |
|---|-------|---------|----------|
| 1 | **Gain Attention** | Focus learner | Hook, surprising fact, question, story |
| 2 | **State Objectives** | Set expectations | "By the end, you will be able to..." |
| 3 | **Recall Prior Learning** | Activate knowledge | Review, connection to previous lesson |
| 4 | **Present Content** | Deliver new info | Explanation, demonstration, examples |
| 5 | **Provide Guidance** | Support learning | Tips, strategies, common mistakes |
| 6 | **Elicit Performance** | Practice | Exercises, activities, application |
| 7 | **Provide Feedback** | Correct/reinforce | Immediate feedback on practice |
| 8 | **Assess Performance** | Verify learning | Quiz, test, demonstration |
| 9 | **Enhance Retention** | Transfer | Summary, real-world application, next steps |

## Lesson Types and Events

### Instruction Lesson
Focus: Events 1-5, 9
- Heavy on presentation and guidance
- Light practice (event 6)

### Demonstration Lesson
Focus: Events 1-4, 6, 9
- Show then do
- Immediate application

### Practice Lesson
Focus: Events 3, 6, 7, 9
- Brief recall, heavy practice
- Lots of feedback

### Assessment Lesson
Focus: Events 3, 8, 9
- Recall prior learning
- Formal assessment
- Transfer guidance

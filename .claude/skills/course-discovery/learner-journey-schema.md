## Journey Stages

1. **Awareness** - Learner realizes they need this
2. **Consideration** - Evaluating options
3. **Enrollment** - Committing to this course
4. **Learning** - Active participation
5. **Completion** - Finishing and applying
6. **Transformation** - Lasting change achieved

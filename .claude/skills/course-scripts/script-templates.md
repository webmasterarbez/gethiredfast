# Script Templates by Content Type

## Video Lesson Script

```markdown
# [Lesson Title]

## HOOK (0:00-0:30)

[VISUAL: Opening shot or attention-grabbing image]

Have you ever [relatable problem]? Today, you'll learn [solution].

## OBJECTIVES (0:30-0:45)

By the end of this lesson, you'll be able to:
1. [Objective 1]
2. [Objective 2]

## SECTION 1: [Topic] (0:45-3:00)

[VISUAL: Diagram or demonstration]

[Script content...]

[B-ROLL: Example of concept in action]

## SECTION 2: [Topic] (3:00-6:00)

[GRAPHIC: Key points on screen]

[Script content...]

[ACTION: Demonstrate on screen]

## SUMMARY (6:00-7:00)

[VISUAL: Recap graphic]

Today we covered:
- [Point 1]
- [Point 2]

Your next step: [Call to action]
```

## Screencast Script

```markdown
# [Lesson Title]

## SETUP

Before recording:
- [ ] Clean desktop
- [ ] Open required apps
- [ ] Prepare example files

## SCRIPT

[ACTION: Open application]

"Let's start by opening [app]..."

[ACTION: Navigate to feature]

"Notice how [observation]..."

[ACTION: Perform task]

"Now I'll [describe action]..."

## COMMON MISTAKES

[ACTION: Show incorrect approach]

"A common mistake is [mistake]. Instead..."
```

## Presentation Speaker Notes

```markdown
# Slide 1: [Title]

SPEAK: Opening hook or question

TRANSITION: "Let's start with..."

---

# Slide 2: [Topic]

KEY POINTS:
- Point 1 (explain with example)
- Point 2 (pause for emphasis)

INTERACT: "Has anyone experienced...?"

---

# Slide 3: [Topic]

SPEAK: [Script for this slide]

DEMO: [If applicable]
```

## Facilitation Guide

```markdown
# [Session Title]

## Pre-Session (15 min before)
- [ ] Test technology
- [ ] Open resources
- [ ] Welcome early arrivals

## Opening (5 min)
- Welcome and housekeeping
- Session objectives
- Icebreaker: [specific activity]

## Content Block 1 (20 min)
TEACH: [Key concept]
ACTIVITY: [Specific exercise]
DEBRIEF: Questions to ask

## Content Block 2 (20 min)
...

## Closing (10 min)
- Recap key points
- Q&A
- Next steps
```
